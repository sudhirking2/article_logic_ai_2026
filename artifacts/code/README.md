# Demo Files for logify2.py Implementation

This directory contains demonstration and testing files for the enhanced logify2.py implementation with Stanford OpenIE integration.

## Files

### **Core Demo Files**

#### `test_logify2_demo.py`
- **Purpose**: Basic OpenIE extraction demo without LLM call
- **Shows**: OpenIE triple extraction and prompt formatting
- **Usage**: `python3 test_logify2_demo.py`
- **Output**: Displays extracted triples and formatted LLM input

#### `logify2_full_demo.py`
- **Purpose**: Complete pipeline demo with simulated LLM response
- **Shows**: Full logify2 workflow from text → OpenIE → enhanced logic
- **Usage**: `python3 logify2_full_demo.py`
- **Output**: Complete logical structure with simulated LLM processing

#### `stanford_openie_demo.py`
- **Purpose**: Stanford CoreNLP OpenIE integration demonstration
- **Shows**: Professional-grade OpenIE extraction with 24 triples from Alice example
- **Usage**: `python3 stanford_openie_demo.py`
- **Output**: Comprehensive Stanford OpenIE results with enhanced logical structure

### **Test Data**

#### `test_text.txt`
- **Purpose**: Alice example text for testing
- **Content**: Complex logical relationships, conditionals, preferences
- **Usage**: Input text for all demo scripts

### **Demo Output Files**

#### `logify2_demo_output.json`
- **Generated by**: `test_logify2_demo.py`
- **Contains**: Basic OpenIE extraction results and formatted LLM input

#### `logify2_full_demo.json`
- **Generated by**: `logify2_full_demo.py`
- **Contains**: Complete simulated logical structure with primitive propositions, constraints

#### `stanford_openie_full_demo.json`
- **Generated by**: `stanford_openie_demo.py`
- **Contains**: Comprehensive Stanford OpenIE results with 24 triples and enhanced logical structure
